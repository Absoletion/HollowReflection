# Archived project resources

These files are historical working outputs kept for reference only. They are
not inputs to the current build or verification path.

- `legacy-art/`: superseded generation rounds, bundle exports, and preview GIFs.
- `legacy-builds/`: disposable standalone HTML snapshots and debug logs.

Active source inputs remain in their original locations. In particular, the
current runtime bundles and the legacy generator inputs referenced by
`tools/` were not moved.
